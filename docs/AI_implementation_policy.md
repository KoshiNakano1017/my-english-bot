# AI自動実装・テスト方針

本プロジェクトでは、AI（大規模言語モデル, LLM）を活用し、以下の開発・テストプロセスを自動化します。

## 1. 実装自動化
- 要件定義・設計ドキュメントをもとに、AIがコードを自動生成
- サービス層・ユーティリティ層の関数やクラスもAIが自動実装
- コードレビューもAIが支援

## 2. テスト自動化
- AIがユニットテスト・統合テストコードを自動生成
- テストケースの網羅性もAIがチェック
- テスト実行・結果検証も自動化

## 3. ドキュメント自動生成
- コードから自動でdocstringやAPI仕様書を生成
- READMEや開発方針もAIが自動作成

## 4. 継続的改善
- テスト失敗時はAIが自動で修正案を提案
- 新機能追加やリファクタリングもAIが主導

---

AI駆動開発により、開発効率・品質向上を目指します。
